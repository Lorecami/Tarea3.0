# Tarea3.0 - Automatización de CI/CD con GitHub Actions y Docker

## Descripción

Este proyecto corresponde a la Tarea 3.0 de automatización de CI/CD utilizando GitHub Actions y Docker.

La aplicación desarrollada se denomina:

**ejercicio:3.0.0**

Es una calculadora simple desarrollada en Python que permite realizar operaciones básicas como suma, resta, multiplicación y división.

## Tecnologías utilizadas

- Python 3.12
- Pytest
- Docker
- GitHub Actions
- GitHub Container Registry
